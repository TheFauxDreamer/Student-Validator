Welcome!

This validator is designed to significantly reduce the amount of manual data checking required for the transition from Integris to Kaartdijin.

Using reports from SIS (only one is mandatory) this site will attempt to flag students and their contacts with the most common errors.

Student Checks

  • Missing Form
  • Missing Year
  • Missing / Valid WASN
  • Valid Address
  • Content in Address Line 3 & 4
  • Admissions Check (Future Entry Date)
  • Check Medical Condition Severity
  • Potential Sibling Mismatches

Contacts Checks

  • Incomplete Name
  • Invalid, duplicate or missing email
  • Missing, duplicate or invalid mobile number
  • Missing gender information (requires XML)
  • Guardian address differs from student address

Features []

  • Medical: Show/hide medical details
  • Colourful Mode: Colour-code student information
  • Show Valid: Include/exclude students without issues
  • WASN: Switch between WASN and SIS num
  • Email MM: Email checks based on Mail Marker
  • Student Resident: Only shows address mismatch for SRs

How to Generate the Reports

Student Mail-merge (CSV)

Takes 2 seconds per student (≈16min per 500)
  1. 1.In SIS open Student Details
  2. 2.Binoculars > click Find (don't change any options here), then click Select
  3. 2b.(Optional) You can include Admissions Students with Advanced Search
  4. 3.Click Yes in the warning (You'll see a red dot in the top left of Student Details)
  5. 4.Click Send Letter > check "Produce a CSV file for use in a subsequent mail-merge"
  6. 5.Name & Save this somewhere you can find it
  7. 6.Wait. Once completed you can drag & drop the report into the top right.

Student Data XML Export (Optional)

Required for gender validation
  1. 1.In SIS open Admin
  2. 2.Click Utilities, find "Student Details XML Export" & double click to open it
  3. 3.Click Find Students, click Find (don't change any options here), click Select
  4. 3b.(Optional) You can include Admissions Students with Advanced Search
  5. 4.In the warning pop-up click yes
  6. 5.Click "Next >>", then click Export
  7. 6.Name & Save this somewhere you can find it
  8. 7.Wait. Once completed you can drag & drop the report into the top right.
